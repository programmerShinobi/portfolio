# Security Policy

## Reporting a vulnerability

This is a static, client-side portfolio with no backend and no user data. If you
nonetheless find a security issue (e.g. a dependency vulnerability or a leaked
secret), please report it privately:

- Email the maintainer at the address listed on the portfolio's contact section.
- Do not open a public issue for sensitive reports.

Please include a description, steps to reproduce, and any relevant links. You can
expect an acknowledgement within a reasonable timeframe.

## Scope

- No secrets are committed to this repository.
- Dependencies are monitored via Dependabot and gated by CI.
